## GDSQL v0.5.0 — Initial Public Release

A Godot 4 plugin that turns SQL-like queries into database operations for ConfigFile-based data storage.

### Features
- SQL-based CRUD: `SELECT`, `INSERT`, `UPDATE`, `DELETE` with familiar syntax
- Database management: create, alter, drop databases and tables via admin API
- Visual query builder with graph-based SQL editor
- GBatis mapper: MyBatis-style XML mapper files for GDScript
- Expression engine with SQL-like expression evaluation
- Aggregation support: `COUNT`, `SUM`, `AVG`, `MIN`, `MAX`, `GROUP_CONCAT`
- Condition chaining with AND/OR for dynamic query building
- Data import/export with password/encryption support
- Multi-language support (en, zh_CN, zh_TW, ja, ko, es, pt, fr, de, ru, it, pl, tr)
- Open source under MIT License

### Notes
- Requires Godot 4.7+
- This is the first public release. APIs are stable but may evolve.
