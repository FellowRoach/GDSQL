# 可视化工作台

GDSQL 工作台作为 Godot 编辑器的独立主屏提供。点击编辑器顶部的 **GDSQL** 按钮即可访问。

## 界面概览

工作台由几个集成面板组成：

- **左侧面板**：数据库树形浏览器
- **中央面板**：表数据 / 结构 / SQL 编辑器（标签页切换）
- **右侧面板**：表检查器 / 详情面板

## 数据库树形浏览器

左侧树形视图以层级结构显示所有数据库及其数据表。

- **展开**数据库查看其数据表
- **点击**数据表在数据编辑器中打开
- **右键点击**打开上下文菜单（创建、删除、重命名等）

## 数据表查看器

数据查看器以 Excel 风格网格显示表内容：

- **滚动**浏览大数据集
- **点击**单元格进行行内编辑——修改立即生效
- **拖拽**列边框调整列宽
- **点击**列头排序

### 行内编辑

点击任意单元格进入编辑模式，支持：

- 直接文本输入
- 针对不同数据类型（int、float、Vector2 等）的类型编辑器
- 按 Enter 或点击其他地方即时提交

### 列宽调整

拖拽列头右边框调整宽度，方便查看长字符串内容。

## 表结构编辑器

切换到 **结构** 标签页查看和修改列定义：

- 列名
- 数据类型
- 默认值
- 注释 / 描述
- 主键和自增设置

## 模式管理

### 创建数据库

使用 **新建数据库** 对话框输入数据库名称。

### 创建数据表

使用 **新建表** 对话框定义：
- 表名
- 列定义（名称、类型、默认值、注释）
- 主键列
- 自增设置

### 删除

在树形视图中右键点击数据库或数据表进行删除。

## SQL 查询编辑器

SQL 编辑器提供专门的空间编写和执行 SQL 查询：

- **语法感知**输入区域
- **执行**按钮运行查询
- **结果面板**以网格显示查询输出
- **查询历史**——自动记录执行过的查询

## 差异视图

对比两个版本的数据表内容。差异视图高亮显示：
- **新增**行（绿色）
- **删除**行（红色）
- **修改**行（黄色）

适合在提交版本控制前审查变更。
