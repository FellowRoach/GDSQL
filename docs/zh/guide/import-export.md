# 导入与导出

GDSQL 支持多种格式的数据导入导出，方便与其他工具互操作。

## 导出格式

### CSV
标准逗号分隔值格式，兼容 Excel、Google Sheets 等电子表格工具。

### JSON
JSON 数组格式，适合 API 集成和数据交换，保留数据类型。

### CFG
Godot 原生 ConfigFile 格式，人类可读，保留所有 Godot 数据类型（Vector2、Color 等）。

## 导入

1. 在工作台中打开目标表
2. 点击 **导入** → 选择格式（CSV / JSON / CFG）
3. 选择文件
4. 导入器按列名或位置映射数据

## 导出查询结果

1. 在 SQL 编辑器中执行查询
2. 在结果面板点击 **导出**
3. 选择格式并保存

## 提示

- 与使用电子表格的设计师共享数据时导出 CSV
- 与 Web 服务或外部工具集成时导出 JSON
- 备份数据或在 GDSQL 项目间迁移时导出 CFG
